# ShopEase - E-Commerce Website

## Overview

ShopEase is a modern and responsive E-Commerce website developed using HTML, CSS, and JavaScript. The project provides a user-friendly online shopping experience with features such as user authentication, product browsing, wishlist management, and shopping cart functionality.

## Features

* User Registration and Login
* Forgot Password Page
* Product Listing and Product Details
* Shopping Cart Functionality
* Wishlist Management
* Responsive Design for Mobile and Desktop
* Search and Navigation Features
* Modern and Interactive User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/shopease.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
